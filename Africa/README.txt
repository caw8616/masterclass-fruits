# Africa
*iOS app written in swift*
*Written using Swift 5*
*Targeting iOS 14.0*
*Coded on Xcode 12*

## Covers the following topics:

1.  iPhone, iPad, Mac applications
1.  Decode JSON file and fetch data
1.  Grid Layout
1.  Codable, extension, and generics
1.  Playing videos
1.  Map with Annotations
1.  Advanced Grid System
1.  Optimization of app's layout for iPad Screens
1.  Transfer iPad app to desktop app with Mac Catalyst
1.  Motion animation created programmatically

*Part of the tutorials offered on the udemy course "SwiftUI Masterclass 2021 - iOS 14 App Development & Swift 5" by Robert Petras*

# Fructus
*iOS app written in swift*
*Written using Swift 5*
*Targeting iOS 14.0*
*Coded on Xcode 12*

## Covers the following topics:

1.  App Icons
1.  Onboarding Screen with Page Tab View
1.  Data Models and Fetching Data
1.  Store the state of the app in App Storage
1.  Displaying an onboarding or home screen
1.  App Life Cycle in SwiftUI 2.0
1.  Link View in SwiftUI 2.0
1.  Group Box View in SwiftUI 2.0
1.  Disclosure View in SwiftUI 2.0
1.  Displaying views dynamically in a List View  with a loop
